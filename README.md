# ProdeskITMission2
Cash-Flow is a simple Salary &amp; Expense Tracker built with HTML, CSS, and JavaScript. It allows users to add income, track expenses, and view remaining balance in real-time. Features include localstorage support, delete functionality, and budget alerts for better financial management.
